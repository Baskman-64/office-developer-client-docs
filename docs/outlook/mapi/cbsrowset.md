# CbSRowSet

  
  
**Applies to**: Outlook 2013 | Outlook 2016

|:-----|:-----|
|Header file:  <br/> |Mapidefs.h  <br/> |
|Related structure:  <br/> |**SRowSet** <br/> |


## Parameters
