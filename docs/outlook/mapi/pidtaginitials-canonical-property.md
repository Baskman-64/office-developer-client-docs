# PidTagInitials Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Contains the initials for parts of the full name of the recipient. 
  
|Property|Value|
|:-----|:-----|
|Associated properties:  <br/> |PR_INITIALS, PR_INITIALS_A, PR_INITIALS_W  <br/> |
|Identifier:  <br/> |0x3A0A  <br/> |
|Data type:  <br/> |PT_UNICODE, PT_STRING8  <br/> |
|Area:  <br/> |Address  <br/> |
   
## Remarks

These properties provide identification and access information for a recipient. They are defined by the recipient and their organization.
