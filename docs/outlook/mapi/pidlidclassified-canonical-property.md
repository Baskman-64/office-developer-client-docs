# PidLidClassified Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Indicates that a message is classified.
  
|Property |Value |
|:-----|:-----|
|Associated properties:  <br/> |dispidClassified  <br/> |
|Property set:  <br/> |PSETID_Common  <br/> |
|Long ID (LID):  <br/> |0x000085B5  <br/> |
|Data type:  <br/> |PT_BOOLEAN  <br/> |
|Area:  <br/> |General messaging  <br/> |
   
## Remarks

For additional information, see [How to Deploy Message Classification for Outlook 2007](https://msdn.microsoft.com/library/5a220424-edd5-4a21-b7fd-8106c23c3b39.aspx)
