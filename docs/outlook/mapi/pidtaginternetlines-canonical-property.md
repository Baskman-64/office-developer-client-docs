# PidTagInternetLines Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Maps to the MIME Lines header.
  
|Property|Value|
|:-----|:-----|
|Associated properties:  <br/> |PR_INTERNET_LINES  <br/> |
|Identifier:  <br/> |0x1034  <br/> |
|Data type:  <br/> |PT_LONG  <br/> |
|Area:  <br/> |MIME  <br/> |
