# PidNameXSharingCapabilities Canonical Property

  
  
**Applies to**: Outlook 2013 | Outlook 2016 
  
Contains a value that indicates the sharing capabilities of the item.
  
|Property|Value|
|:-----|:-----|
|Friendly names:  <br/> |None  <br/> |
|Property set:  <br/> |PS_INTERNET_HEADERS  <br/> |
|Property name:  <br/> |X-Sharing-Capabilities  <br/> |
|Data type:  <br/> |PT_UNICODE  <br/> |
|Area:  <br/> |Sharing  <br/> |
   
## Remarks

This property contains a value that indicates the sharing capabilities of the message object.
