# PROP_SMTP_USER

Represents the user name for the SMTP account.
  
## Quick info

|Property |Value |
|:-----|:-----|
|Identifier:  <br/> |0x0204  <br/> |
|Property type:  <br/> |PT_UNICODE  <br/> |
|Property tag:  <br/> |0x0204001F  <br/> |
|Access:  <br/> |Read-only  <br/> |
