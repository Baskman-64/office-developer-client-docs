# PROP_INET_PASSWORD

Represents the user password for a general Internet mailbox.
  
## Quick info

|Property |Value |
|:-----|:-----|
|Identifier:  <br/> |0x0102  <br/> |
|Property type:  <br/> |PT_UNICODE \| SECURE_FLAG  <br/> |
|Property tag:  <br/> |0x0102801F  <br/> |
|Access:  <br/> |Read-only  <br/> |
