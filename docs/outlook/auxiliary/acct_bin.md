# ACCT_BIN

A variable of this data type holds a binary value.
  
## Quick info

```cpp
typedef struct { 
    DWORD cb; 
    BYTE * pb; 
} ACCT_BIN; 

```

## Members

_cb_
  
> Number of bytes that _pb_ points to.

_pb_
  
> Pointer to binary information.
