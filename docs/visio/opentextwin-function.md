# OPENTEXTWIN Function

Opens the shape's text block so that the text can be edited.
  
## Syntax

OPENTEXTWIN( )
  
## Remarks

In earlier versions of Visio, this function appears as _OPENTEXTWIN. Visio versions 4.0 and later accept either style.
