# PNTY Function

Returns the _y_-coordinate of a point.
  
## Syntax

PNTX(* _point_** )
  
### Parameters

|**Name**|**Required/Optional**|**Data Type**|**Description**|
|:-----|:-----|:-----|:-----|
| _point_ <br/> |Required  <br/> |**Number** <br/> |The _y_-coordinate of the point. |

### Return value

Number
  
## Example

PNTY(PNT(7,12))
  
Returns 12.
