# DATA3 Function

Returns the text from the **Data 3** box in the **Shape Name** dialog box for a shape (on the **Developer** tab, click **Shape Name** ), as a string. 
  
## Syntax

DATA3 ()
  
## Remarks

If the box is empty, the function returns an empty string.
