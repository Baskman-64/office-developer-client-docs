# OPENSHEETWIN Function

Opens the ShapeSheet window in a new window.
  
## Syntax

OPENSHEETWIN( )
  
## Remarks

In earlier versions of Visio, this function appears as _OPENSHEETWIN. Visio versions 4.0 and later accept either style.
