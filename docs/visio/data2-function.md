# DATA2 Function

Returns the text from the **Data 2** box in the **Shape Name** dialog box for a shape (on the **Developer** tab, click **Shape Name** ), as a string. 
  
## Syntax

DATA2 ()
  
## Remarks

If the box is empty, the function returns an empty string.
